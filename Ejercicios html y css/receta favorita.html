Receta para hacer pasta

  • Siempre usar para la preparación abundante agua, teniendo como referencia una olla casi llena por cada libra de pasta.
  • Agregar la sal al agua una vez que esté hirviendo.
  • Incorporar la pasta y remover para evitar que se pegue.
  • Cocinar según las instrucciones del paquete, probando la pasta para asegurarse de que esté al dente.
  • Escurrir la pasta y reservar un poco del agua de cocción.
  • Mezclar la pasta con la salsa deseada, añadiendo un poco del agua de cocción si es necesario para ajustar la consistencia.

    • Ingredientes:

    • 500 grs de Tallarines de Pastas Doria
    • 2 cucharadas de sal
    • 50cc Aceite de Oliva
